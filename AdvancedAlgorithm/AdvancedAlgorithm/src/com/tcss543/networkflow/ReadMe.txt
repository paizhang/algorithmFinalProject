To run this program, two arguments should be given. 
The first one is the name of the algorithm that will be run("FFA", "SFFA" or "PPA"). 
The second one is the full path of the data set file which will be input. 
This argument is optional. If it is blank, it will load the default data set. 

The program will print out the max flow of given graph. Or it will print out the 
error information.
